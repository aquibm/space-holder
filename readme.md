# space-holder
Weekend project to build a space-themed image placholder service.

## Demo
[Click here for a random space image!](https://place.aquibm.com/720/480)

## Routes

| Path                | Description                                                         |
| ------------------- | ------------------------------------------------------------------- |
| /ping               | Responds with 'pong' if the server is operational                   |
| /{width?}/{height?} | Responds with a random image with the specified optional dimensions |
